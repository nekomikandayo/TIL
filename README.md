### 4月4日
【やったこと】<br>
React学習:'Reactとは'の学習<br>
【一言で言うと】<br>
Reactとは、UIを構築するためのJavaScriptライブラリであり、状態に応じて画面を自動的に更新する宣言的UIが特徴です。<br>
【学び・Why】<br>
従来のjQueryなどはDOMを直接操作するため、状態管理が複雑になりやすい<br>
Reactは状態（state）を中心にUIを構築するため、コードの可読性・保守性が高い<br>
仮想DOMにより、変更差分のみを更新するためパフォーマンスが良い<br>
コンポーネント単位で分割できるため、再利用性が高くチーム開発に向いている<br>
